📌 Project Title

Sales Data Analysis for a Retail Store (Power BI Dashboard)

🎯 Project Overview

This project presents an interactive Power BI dashboard built to analyze retail sales performance across regions, categories, and cities. It provides actionable insights into revenue, profit, and product performance using real-world business intelligence techniques.

🚀 Key Features
Interactive dashboard with slicers (City, State, Sub-category)
KPI cards for Total Sales and Total Profit
Region-wise sales comparison
Category-wise contribution analysis
Geographic visualization using map
Drill-down capabilities for deeper insights
📊 Dashboard Insights
💰 Total Sales: 230M
📈 Total Profit: 286.4K
🥇 Top performing region: West (0.73M sales)
📉 Lowest performing region: South (0.39M sales)
📦 Technology category contributes the highest share
📈 Visualizations Used
KPI Cards
Bar Chart (Sales by Region)
Pie Chart (Sales by Category)
Map Visualization (City-level sales)
Slicers for filtering
🛠️ Tools & Technologies
Power BI
Data Modeling

Data Cleaning
